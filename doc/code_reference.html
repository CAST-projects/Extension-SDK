get_annotations()¶

Returns the annotations associated with the object.

A list of annotations is returned.

Return type:list of tuple (cast.analysers.Type,dict)

An annotation is a tuple (type,parameters) where:

  • type is a Type object
  • parameters is a dictionary with the actual parameters
get_project()¶

Returns the root project (in the sense of enlighten) of the object.

If the object is shared among several projects, return one.

Return type:cast.analysers.Object

save_violation(rule, bookmark, additional_bookmarks=None)¶

rule is the name of a Metamodel property

Parameters:

  • bookmark (cast.analysers.Bookmark) – a bookmark to indicate the position of the violation
  • additional_bookmarks (cast.analysers.Bookmark) – additional bookmarks that help understanding the violation

An object created through plugin.

This class is designed to be used as is or subclassed for your own need.

Type must respect some constraints for the object to be saved. ...

External link API¶

API for External Links.

External Links allow to retreive server objects from other analysis marked as dependency.

It is basically used for creating client to server links.

For example one can retreive database Tables, View etc...

cast.analysers.external_link.find_objects(name, category_name)¶

Find all the external objects of a given name and category.

Parameters:
  • name – name of the object
  • category_name – name of a metamodel category
Return type:

list of cast.analysers.ExternalObject

Example :

# find all tables named ‘Author’:

tables = find_objects(‘Author’,’Database Table’)

@rtype: [cast.analysers.ExternalObject]

cast.analysers.external_link.analyse_embedded(text)¶

Analyse an embedded sql query.

Return type:list of cast.analysers.external_link.EmbeddedResult

New in version 7.3.6.

Tests API Reference¶

This module provide testing help for developping plugins.

Simple usage:

import cast.analysers.test

analysis = cast.analysers.test.DotNetTestAnalysis()

analysis.add_selection(‘path to .csproj’)

analysis.run()

Parameters:path (str) – selection path to be added.
